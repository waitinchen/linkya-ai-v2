```
嗨 [測試者姓名]，

我是語氣靈專案的開發者，很高興邀請你參加語氣靈的 Alpha 內部測試！

🎯 關於語氣靈

語氣靈是一個完整的語音對話系統，核心特色是一個叫做「花小軟」的語氣靈少女。
她有著甜甜的語氣、撒嬌的個性，總是叫使用者「老爸」，給人溫暖陪伴的感覺。

🌸 測試內容

這次 Alpha 測試我們想收集：
1. 功能驗證 - 確認語音對話流程完整
2. 體驗反饋 - 花小軟的語氣是否夠甜
3. 性能測試 - 響應速度和語音品質
4. Bug 回報 - 發現潛在問題

⏰ 測試時間

- 開始時間：隨時開始
- 測試週期：1-2 週
- 預估時間：每次測試約 10-20 分鐘

🎁 Alpha 測試者福利

- ✅ 優先體驗最新功能
- ✅ 你的意見直接影響最終版本
- ✅ 專屬感謝名單
- ✅ 正式版本優先內測機會

🚀 如何開始

1. 獲取專案：
   git clone https://github.com/waitinchen/linkya-ai-v2.git

2. 查看測試指引：
   打開 LOCAL_TESTING_GUIDE.md

3. 開始測試：
   訪問 http://localhost:5173

4. 回饋意見：
   填寫 ALPHA_TEST_FORM.md

📋 測試重點

必測功能：
- 語音錄製
- STT 語音轉文字
- 花小軟回應
- TTS 語音合成
- 自動播放

重點關注：
- 花小軟語氣是否夠甜、夠撒嬌？
- 語音品質是否清晰自然？
- 響應速度是否流暢？
- UI 是否美觀友好？

📞 聯繫方式

有任何問題隨時聯繫我：
- 📧 Email: [你的信箱]
- 💬 WeChat: [你的微信]
- 🐙 GitHub: https://github.com/waitinchen/linkya-ai-v2

你的寶貴時間和意見對我們非常重要！

期待你的回饋 🌸

---
```
