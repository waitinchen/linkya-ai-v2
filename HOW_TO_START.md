# 🎯 如何啟動語氣靈

## ✅ 已啟動後端！

後端服務已經啟動並運行在 **http://localhost:3000**

## ⏳ 正在啟動前端...

前端正在安裝依賴，請稍候 1-2 分鐘

---

## 📋 檢查清單

### ✅ 後端狀態

請檢查 **後端終端窗口**，應該看到：
```
🚀 語氣靈後端運行於 http://localhost:3000
✅ STT 路由: /api/stt
✅ LLM 路由: /api/llm
✅ TTS 路由: /api/tts
```

### ⏳ 前端狀態

請檢查 **前端終端窗口**，應該看到：
1. 正在安裝依賴（`npm install` 運行中）
2. 安裝完成後會自動啟動 Vite
3. 最終應該顯示：
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:5173/
```

---

## 🌐 訪問地址

前端啟動後，訪問：**http://localhost:5173**

---

## ⏰ 等待時間

- 後端啟動：1-2 秒 ✅
- 前端安裝依賴：1-2 分鐘 ⏳
- 前端啟動：2-3 秒

---

## 🎉 完成後

一旦看到 Vite 顯示 `ready`，就可以訪問：

**http://localhost:5173**

開始與花小軟對話吧！🌸

---

## 🆘 如果前端啟動失敗

如果前端終端顯示錯誤，請手動在新終端執行：

```powershell
cd C:\Users\waiti\linkya-ai-v2\soft-voice-mvp\frontend
npm install
npm run dev
```

**請查看終端窗口，前端應該很快就會啟動完成！** ⏳
